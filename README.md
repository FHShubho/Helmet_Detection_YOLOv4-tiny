# Helmet_Detection_YOLOv4-tiny

> Detection of helmet using YOLOv4-tiny model



### Dataset

- [Helmet Detection](https://www.kaggle.com/andrewmvd/helmet-detection)
- Additional images publicly available on web.



### Model

- mAP = 98.1%
